Konfigurasi JSONBin.io
💡 Buat akun gratis di jsonbin.io → Buat Bin baru → Salin Bin ID dan Master Key di sini. Landing page akan membaca data dari Bin ini secara otomatis.
Cara setup di Landing Page:
// Di index.html, tambahkan: const JSONBIN_ID = "PASTE_BIN_ID_DISINI"; const JSONBIN_KEY = "PASTE_ACCESS_KEY_DISINI"; // Halaman akan auto-fetch saat load
